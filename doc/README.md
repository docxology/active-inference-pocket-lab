# doc/

Comprehensive documentation for Spin — The Active Inference Pocket Lab.

## The Seed

> This project began with a single creative brief. The document [**initial-pocket-lab-prompt.md**](initial-pocket-lab-prompt.md) is the original seed from which every architectural decision, design token, and interaction pattern in Spin was grown. It defines the constitutional narrative ("the Vibe"), the 6 core rules, the Triple-Stream Architecture ("the Helix"), and the mandate for Module 1: The First Orbit. **This file is preserved exactly as-written** — it is the project's founding document and should be treated as such.
