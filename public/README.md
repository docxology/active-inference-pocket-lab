# public/

Static assets served by Vite at the root URL.

## Files

| File | Description |
| --- | --- |
| `spin-icon.svg` | Spin logo: concentric orbit rings with colored dots (Pulse red, Vision purple, Core teal, Hearth amber) |

## Usage

Files in `public/` are served as-is without processing. Reference them with absolute paths:
```html
<link rel="icon" href="/spin-icon.svg" />
```
